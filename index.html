PERJALANAN PENGEMBANGAN WEBSITE SAYA

Selamat datang di Blog pembuatan Website Farhan Dev

Tahap 1: Perencanaan & Pengaturan HTML

Oktober 10, 2024

Perjalanan saya dimulai dengan merencanakan struktur website saya. Saya membuat file `index.html` dasar untuk menyusun bagian seperti header, konten utama, dan footer. Langkah ini berfokus pada pembuatan kerangka HTML yang bersih untuk menampung semua konten.

Tahap 2: Penataan dengan CSS

Oktober 12, 2024

Selanjutnya, saya menambahkan CSS untuk mengatur setiap bagian situs. Ini termasuk merancang tata letak responsif, memilih warna, dan menetapkan tipografi agar sesuai dengan gaya pribadi saya. Saya fokus menciptakan tampilan yang konsisten untuk portofolio saya.

Tahap 3: Menambahkan JavaScript untuk Interaktivitas

Oktober 15, 2024

Saya kemudian menambahkan JavaScript untuk membuat situs lebih dinamis. JavaScript membantu menambahkan interaktivitas, seperti efek hover dan aksi tombol, yang membuat website terasa lebih menarik.

Tahap 4: Pengujian Responsivitas

Oktober 18, 2024

Memastikan situs terlihat bagus di semua perangkat sangat penting. Saya mengujinya di berbagai ukuran layar dan menyesuaikan CSS sesuai kebutuhan agar sepenuhnya responsif, terutama untuk pengguna ponsel.

Tahap 5: Mempublish proyek ke Netlify

Oktober 31, 2024

Langkah terakhir adalah mendeploy situs saya. Saya memilih Netlify karena kemudahannya. Setelah mengunggah file saya langsung ke Netlify, website saya sudah bisa diakses di link ini https://frhndevweb.netlify.app.
